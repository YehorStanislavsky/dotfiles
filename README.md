# .config

файлы конфигураци

### vim install 

```
sudo apt install vim -y && echo -e "set number\nset cursorline\nsyntax" on > .vimrc
```
 ### dropbox 
 ```
 sudo apt install -y dropbox
 ```
 
 ### flatpak 
 ```
sudo add-apt-repository ppa:flatpak/stable
sudo apt update
sudo apt install flatpak
    
 ```
